セットアップガイド

広告クリック → LINE友だち追加の計測を始めるまでの手順です。所要15〜30分。

いちばん簡単な方法：ダッシュボード上部の「かんたんスタート（LINE連携ウィザード）」の案内どおりに進めれば、キー入力・接続テスト・Webhook設定・初期設定はすべて自動でご案内します。このガイドは、その前準備（STEP 1）と補足のためのものです。

全体の流れ

  1. LINE公式アカウントを作成し、Messaging APIを有効化してキーを用意（← このSTEP 1だけLINE側での作業）
  2. ダッシュボードのウィザードにキーを貼付 → 接続テスト
  3. ウィザードのWebhook URLをLINEに貼付 → 受信確認
  4. ウィザードで初期設定を一括適用（あいさつ・応答・ステップ配信・会話ボット）
  5. （任意）Meta / TikTok のピクセル登録・計測リンク発行 → テスト

STEP 1. LINE公式アカウントとMessaging APIを用意（キーの発行）

1-1. LINE公式アカウントを作成

LINE公式アカウントの開設ページ から、院のLINE公式アカウントを作成します（無料）。すでにお持ちの場合はそのまま次へ。

1-2. Messaging APIを有効化して「Channel ID / Channel secret」を取得

  1. LINE公式アカウントマネージャー を開く
  2. 右上の 設定 → 左メニューの Messaging API → 「Messaging APIを利用する」
  3. プロバイダーを選択（無ければ院名で新規作成）→ 利用規約に同意
  4. 有効化されると Channel ID と Channel secret が表示されます

1-3. 「チャネルアクセストークン（長期）」を発行

同じ画面の案内から LINE Developersコンソール を開き、Messaging API設定 タブの「チャネルアクセストークン（長期）」で 発行 → コピーします。

用意する値は ①Channel secret と ②チャネルアクセストークン（長期） の2つです。あとはウィザードに貼るだけ。

1-4. LINE側の応答設定（推奨）

  • 応答モード：Bot／あいさつメッセージ：オフ（Keiroが挨拶を送るため）／応答メッセージ：オフ推奨／Webhook：オン

STEP 2. ウィザードでキーを入力 → 接続テスト

ダッシュボード上部のウィザード「② キーを入力して接続する」に、STEP 1で用意した Channel secret と アクセストークン を貼り付けて「保存して接続テスト」を押します。「✓ 接続できました」と出れば成功です（トークンが正しいか自動で確認します）。

STEP 3. ウィザードのWebhook URLをLINEに設定 ＆ 初期設定

ウィザード「③ Webhookを設定する」に表示される あなた専用のWebhook URL をコピーし、LINE DevelopersのMessaging API設定の Webhook URL に貼り付けて「検証」→「Webhookの利用」をオンにします。ウィザードで「受信できたか確認」を押し、「✓ 受信を確認」が出ればOK。続けて「④ 初期設定を一括適用」を押すと、あいさつ・自動応答・ステップ配信・会話ボットがまとめて作成されます。

STEP 4.（任意）広告ピクセルを登録

Meta（Facebook/Instagram広告）

Meta Events Manager で ピクセルID と Conversions APIのアクセストークン を取得し、Keiroの連携設定に入力します。友だち追加が成立すると Lead イベントが自動送信されます。

TikTok広告

TikTok Events Manager で ピクセルID と Events API アクセストークン を取得し入力します。CompleteRegistration が送信されます。

※ 入力した院だけに送信されます。未入力の媒体には送信されません。

STEP 5. 計測リンクを発行して広告に設定

ダッシュボードの「計測リンクを作成」で、名前・友だち追加URL・送信媒体（例 meta）を入れて作成します。表示される 計測URL（https://……/c/＜ID＞）を、広告の遷移先URLに設定してください。

このURLをタップすると、計測した上で認証画面を挟まずLINEの友だち追加へ転送されます。

STEP 6. テストして確認

  1. スマートフォンで計測URLを開く → LINE友だち追加
  2. 届いた挨拶メッセージのリンクを1回タップ（これで紐づけが確定します）
  3. ダッシュボードの「直近の友だち追加」に 紐づけ済 が表示されればOK

紐づけの仕組み（重要）

スマホでは「広告クリック時のブラウザ」と「LINEアプリ内ブラウザ」が別物のため、主にIPアドレスと時間で紐づけます。誤った計測を避けるため、確実でない場合はあえて紐づけません（精度優先）。Wi-Fiとモバイル回線の切り替え直後などは紐づかないことがあります。

困ったとき

  • Webhook検証に失敗する → Channel Secretの入力ミス、またはWebhook URLの貼り間違いをご確認ください。
  • 挨拶が届かない → Channel Access Tokenの未入力、またはLINE側の「応答モード=Bot」をご確認ください。
  • 紐づかない → 挨拶リンクをタップしているか、計測URL（/c/…）から友だち追加しているかをご確認ください。
