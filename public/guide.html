セットアップガイド

広告クリック → LINE友だち追加の計測を始めるまでの手順です。所要15〜30分。

先に管理画面（ダッシュボード）にログインしておいてください。以下の「Webhook URL」「計測リンク」は管理画面で確認・発行できます。

全体の流れ

  1. LINEのMessaging APIを用意（チャネル作成）
  2. KeiroにLINEの情報を登録し、LINEにWebhook URLを設定
  3. （任意）Meta / TikTok のピクセルを登録
  4. 計測リンクを発行して広告に設定
  5. テストして紐づけを確認

STEP 1. LINE Messaging API を用意

1-1. チャネル作成

LINE Developers でプロバイダーを作成し、対象のLINE公式アカウントに紐づく Messaging API チャネル を作成します。

1-2. 必要な値を取得

  • チャネルシークレット（Basic settings タブ）
  • チャネルアクセストークン（長期）（Messaging API タブで発行）

1-3. LINE公式アカウントの設定

  • 応答モード：Bot
  • あいさつメッセージ：オフ（Keiroがclaimリンク付きの挨拶を送るため）
  • 応答メッセージ：オフ推奨
  • Webhook：オン

STEP 2. Keiroに登録 ＆ Webhook設定

2-1. Keiroの「連携設定」に入力

ダッシュボードの「連携設定」で、LINE Channel Secret と LINE Channel Access Token、友だち追加URL（https://lin.ee/xxxx 形式）を保存します。トークンは暗号化保存され、画面には再表示されません。

2-2. Webhook URL をLINEに設定

連携設定に表示される あなた専用のWebhook URL（https://……/webhook/＜あなたのトークン＞）をコピーし、LINE DevelopersのMessaging API設定の Webhook URL に貼り付けて「検証」を押します。成功すればOKです。

STEP 3.（任意）広告ピクセルを登録

Meta（Facebook/Instagram広告）

Meta Events Manager で ピクセルID と Conversions APIのアクセストークン を取得し、Keiroの連携設定に入力します。友だち追加が成立すると Lead イベントが自動送信されます。

TikTok広告

TikTok Events Manager で ピクセルID と Events API アクセストークン を取得し入力します。CompleteRegistration が送信されます。

※ 入力した院だけに送信されます。未入力の媒体には送信されません。

STEP 4. 計測リンクを発行して広告に設定

ダッシュボードの「計測リンクを作成」で、名前・友だち追加URL・送信媒体（例 meta）を入れて作成します。表示される 計測URL（https://……/c/＜ID＞）を、広告の遷移先URLに設定してください。

このURLをタップすると、計測した上で認証画面を挟まずLINEの友だち追加へ転送されます。

STEP 5. テストして確認

  1. スマートフォンで計測URLを開く → LINE友だち追加
  2. 届いた挨拶メッセージのリンクを1回タップ（これで紐づけが確定します）
  3. ダッシュボードの「直近の友だち追加」に 紐づけ済 が表示されればOK

紐づけの仕組み（重要）

スマホでは「広告クリック時のブラウザ」と「LINEアプリ内ブラウザ」が別物のため、主にIPアドレスと時間で紐づけます。誤った計測を避けるため、確実でない場合はあえて紐づけません（精度優先）。Wi-Fiとモバイル回線の切り替え直後などは紐づかないことがあります。

困ったとき

  • Webhook検証に失敗する → Channel Secretの入力ミス、またはWebhook URLの貼り間違いをご確認ください。
  • 挨拶が届かない → Channel Access Tokenの未入力、またはLINE側の「応答モード=Bot」をご確認ください。
  • 紐づかない → 挨拶リンクをタップしているか、計測URL（/c/…）から友だち追加しているかをご確認ください。
